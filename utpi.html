Utility/Tools Programming Interface
 



Welcome to the UTPI main page !

UTPI is a free API for your Unity projects offering you free unity packages with tools and objects to help you save time while working.
UTPI is updated constantly. Feels free for you to suggest me an improvement to the existing tools ! You can contact me by email at capelier.nicolas@gmail.com.

Download

You can download every asset, utility and tool in a single package or download them separately on their page.

Scripting Reference

[A]ssets, [T]ools and [U]tilities are grouped by packages.

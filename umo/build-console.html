Build Console

The BuildConsole class allows you to create a console in your game's build and display logs in it.

Download

Download Package

Public Methods

Log(object message) Logs a message to the Build Console.
LogWarning(object message) A variant of BuildConsole.Log that logs a warning message to the console.
LogError(object message) A variant of BuildConsole.Log that logs an error message to the console.
Assert(bool condition) Assert a condition and logs an error message to the build console on failure.
Assert(bool condition, object message) Assert a condition and logs an error message to the build console on failure.
Show() Displays the build console.
Hide() Hides the build console.
SetConsoleSize(int size) Changes the console's maximum displayed logs. Warning : SetConsoleSize() resets the console's logs memory. Default size is 10.
SetFont(Font font) Changes the font in the build console. Default font is Arial.
SetColor(Color color) Changes the text color in the build console. Default color is black.
SetFont(int size) Changes the text size in the build console. Default size for a full HD screen is 40.

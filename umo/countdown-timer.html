Countdown Timer

The CountdownTimer class allows you to create simple and efficient timers.

Download

    Free download will be avaible soon.

Constructor

CountdownTimer() Creates a countdown timer that will not tick until you give it a new time.

Action

Action action Can not be null. This action is called when the countdown timer hits zero.

Public Methods

MethodName(ParamType param) ReturnType Description
SetTime(float time) CountdownTimer Set a new time to the countdown timer.
Returns: The countdown timer.
SetTime(float time, Action action) CountdownTimer Set a new time to the countdown timer.
Returns: The countdown timer.
SetTime(float time, bool startPaused) CountdownTimer Set a new time to the countdown timer.
Returns: The countdown timer.
SetTime(float time, Action action, bool startPaused) CountdownTimer Set a new time to the countdown timer.
Returns: The countdown timer.
SetAction(Action action) CountdownTimer Set a new action to the countdown timer.
Returns: The countdown timer.
Replay() CountdownTimer Replay the countdown timer with the same parameters.
Returns: The countdown timer.
GetTime() float Returns: The current time of the countdown timer.
Pause() void Pause the countdown.
Resume() void Resume the countdown.
IsPaused() bool Returns: True if the countdown timer is paused.
IsFinished() bool Returns: True if the countdown timer is finished.
End() bool End the countdown timer and call the action.
Returns: True if the countdown timer was not already finished and the action is not null.
Cancel() bool Cancel the countdown timer without calling the action.
Returns: True if the countdown timer was not already finished.

Unity Modules
 



Lerper

The Lerper class allows you to create a lerp overtime.

Download

    Free download will be avaible soon.

Constructor

Lerper() Creates a lerper that will not tick until you call the Lerp or Replay methods.

Actions

Action<float> lerpAction Can not be null. This action is called on every frame when the lerper is running. The float variable is the lerped value.
Action completedAction Optional. This action is called when the lerper finishes the lerp.

Public Methods

MethodName(ParamType param) ReturnType Description
Lerp(float a, float b, float time, Action<float> lerpAction) Lerper Start a new lerp.
Returns: The lerper.
Lerp(float a, float b, float time, Action<float> lerpAction, Action completedAction) Lerper Start a new lerp.
Returns: The lerper.
SetLerpAction(Action<float> lerpAction) Lerper Set a new lerpAction to the lerper.
Returns: The lerper.
SetCompletedAction(Action completedAction) Lerper Set a new completedAction to the lerper.
Returns: The lerper.
SetActions(Action<float> lerpAction, Action completedAction) Lerper Set both lerpAction and completedAction of the lerper.
Returns: The lerper.
Replay() Lerper Replay the lerper with the same parameters.
Returns: The lerper.
GetTime() float Returns: The current progression time of the lerper.
GetCompletion() float Returns: The current completion (between 0 and 1) of the lerper.
Pause() void Pause the lerp.
Resume() void Resume the lerp.
IsPaused() bool Returns: True if the countdown lerper is paused.
IsFinished() bool Returns: True if the lerper is completed.
End() bool End the lerper and call the completedAction.
Returns: True if the lerper was not already finished and the lerpAction is not null.
Cancel() bool Cancel the lerper without calling the completedAction.
Returns: True if the lerper was not already finished.

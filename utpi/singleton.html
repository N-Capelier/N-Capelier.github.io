Singleton

The singleton design pattern restricts the instantiation of a class to a single instance with an easy acces to it.

Download

Download Package

Instantiation

To transform your class to a singleton, you first need to make it inherit from the Singleton<T> class with T the name of your class.
Then you need to use the method CreateSingleton() (generally in the Awake() method) to make your class an actual singleton.
Now you can acces your class from everywhere with MyClassName.Instance.
⚠ If your class is instantiated more than once, the other instances will be destroyed !

Properties

T Instance {get} Gets the instance of the Singleton.

Public Methods

CreateSingleton() Makes the object a Singleton.
CreateSingleton(bool isUndestroyableOnLoad) Makes the object a non-destroyable on load Singleton.

Clocks

The Clock class allows you to create simple and efficient timers.

Download

Download Package

Instantiation

The Clock class uses a personnal Update() method on an invisible DontDestroyOnLoad game object called [Update Caller].
This is why you need to instantiate your Clock at the begining of the behaviour of your script (Generally in the Start() method) after declaring it at the top of your script.

Constructors

Clock() Creates a clock that will not tick until you give it a new time.
Clock(float time) Creates a clock ticking until the time hits zero.
Clock(float time, bool startsPaused) Same as previous + the clock will start paused and will wait to be set on "play" to tick.

Properties

float time {get} Current time of the clock.
bool onFinish {get} Returns true when the clock hits zero.
bool finished {get} Returns true if the clock's timer is equal to zero.
bool paused {get} Returns true if the clock is paused.

Public Methods

SetTime(float time) Gives a new time to the clock.
Stop() Stops the clock and set its time to zero.
Play() Unpauses the clock.
Pause() Pauses the clock.
